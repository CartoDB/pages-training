Builder 101: Making an Interactive Hurricane Map

This is a lesson in how to make an interactive hurricane map in CARTO

Final product of this lesson will look like this

Map Setup

  • Log into CARTO
  • Open up Maps Dashboard

Create a New map

  • Select Create Empty Map

You will see a basic map with no data on it and a layer card with a warning symbol, telling you there is no geometry (no shapes, points or lines on the map)

  • Double click on the title of the map to make it active
  • Name the map Interactive Hurricane Map

  • Change Basemap to put labels below

Open up the basemaps sidebar by clicking the current basemap Select Positron (labels below) as the new basemap.

Add Hurricane Zones

  • Download Hurricane Zones (link)
  • Hit Download button on interface

  • Choose SHP as format
  • In your map, click Add button on top of layer sidebar

  • Go to the Connect Dataset tab
  • Drag and Drop file to the upload field.
  • Click Add Layer
  • Double click layer name and rename to Hurricane Zones
  • Click 3-dot icon and Center map on layer

Style Hurricane Zones

  • Click on layer to go into layer settings
  • Select the Style tab (may already be selected)
  • Click on the Fill bar and select the right-hand By Value setting, then select the value (column) hurricane_

Once you select that, it will assign colors to your hurricane zones.

Use actual NYC Hurricane Zone colors

  • In the bottom of the style panel, flip this switch to edit the CartoCSS in the backend

  • There you will see the underlying code to edit the current colors to the hurricane colors

  • Replace the code there with this:

     #layer {polygon-fill: ramp([hurricane_], (#bababa, #dc7e5e, #e39d5d, #e9e066, #bed066, #98c576, #52A696), ("0", "1", "2", "3", "4", "5", "6"), "=");
  polygon-opacity:1;
  line-width: 0;
  line-color: #FFF;
  line-opacity: 1;
}

Add Evacuation Centers

  • Evacuation Centers (link)
  • Hit Download button on interface
  • Choose CSV as format
  • Click Add button on top of layer sidebar
  • Go to the Connect Dataset tab
  • Drag and Drop file to the upload field.
  • Click Add Layer
  • Double click layer name and rename Evacuation Centers

Style Evacuation Centers

  • Select POINT SIZE and change it to 19.5
  • Select the color bar next to the number and make it red
  • Select STROKE SIZE and change it to 5

Analysis

Generate a 20-minute walk from each evacuation center

Click on Evacuation Centers layer name or layer letter Go to Analysis tab

  • Select Add Analysis
  • Select analysis Create Travel/Distance Buffers
  • Click Add Analysis
  • Enter the following parameters

Option 1 (credits used)

  • Type: Time
  • By: walk
  • Time (Seconds): 1800
  • Tracts: 1
  • Boundaries: intact

Option 2 (credits not used)

  • Type: Distance
  • Units: mi
  • Radius: .5
  • Tracts: 1
  • Boundaries: intact

Press return. Analysis will run, generating a 20-minute walk (Option 1) or a 5 mile circle around each center (Option 2).

Separate evacuation centers from polygon

  • Return to Top level of map
  • Find the Evacuation Centers Layer
  • Click and hold the A1 module in the layer card
  • Drag analysis results C1 into its own layer
  • Double click new layer name and rename Evacuation Range

Style Evacuation Range layer

  • Select Fill and set the opacity to .07 towards the bottom, either by changing the number or using the slider.
  • Set the stroke to 1

Add Greenpoint Buildings

  • Buildings in Greenpoint (link)
  • Hit Download button on interface
  • Choose .gpkg as format
  • Click Add button on top of layer sidebar
  • Go to the Connect Dataset tab
  • Drag and Drop file to the upload field.
  • Click Add Layer

Style buildings by zone

  • In the bottom of the style panel, flip this switch to edit the CartoCSS in the backend:

There you will see the underlying code to edit the current colors to the hurricane colors

  • Replace the code there with this:

    #layer {polygon-fill: ramp([zone], (#bababa, #dc7e5e, #e39d5d, #e9e066, #bed066, #98c576, #52A696), ("0", "1", "2", "3", "4", "5", "6"), "=");
  polygon-opacity:1;
  line-width: 0;
  line-color: #FFF;
  line-opacity: 1;
}

Lower opacity of the hurricane zones

  • Go to layer : Hurricane zones
  • Style > Cartocss > edit line: polygon-opacity:.4;

Add popup to Buildings

  • Go to layer : Greenpoint Buildings
  • Tab > Pop-up > Activate a popup (any color)
  • Turn on: binid, height roof, numfloors, year_built

Make link to BIN site open in DOB website:

  • Go to layer: Greenpoint Buildings
  • Popup > HTML

  • Find where {{bin_id}} is located, replace that H4 block (line 9, probably) with this code: 

    <h4 class="CDB-infowindow-title"><a href="http://a810-bisweb.nyc.gov/bisweb/PropertyProfileOverviewServlet?bin={{bin_id}}" target=blank_</a>{{bin_id}}</h4>

  • Click on a building and activate BIN link

Add widgets

  • On top level, switch to the Widgets panel
  • Select Add Widget

You'll see four tabs, Category, Histogram, Formula and time series Check the following boxes:

  • Histogram > heightroof, num_floors
  • Formula > point count or polygon count
  • Timeseries > yearbuiltdate

Style Widgets

On all:

  • Fill color: #ff0000
  • Rename all widgets
  • Create a custom color ramp:
    • On a histogram, select Custom color at bottom
    • Assign a range of colors to be activated in autostyling

Join zones to buildings

  • Click into layer, and click table name hyperlink below file name.
  • Add a column to the greenpoint_buildings file named zone
  • Add column
  • Name zone_id

  • Open SQL pane at bottom

    Update greenpoint_buildings SET zone_id = hurricane_ from hurricane_zones WHERE ST_Intersects(hurricane_zones.the_geom, greenpoint_buildings.the_geom)

This is the base code for that: 

Update {table} SET {new_column} = {source_column) from {source_table} WHERE ST_Intersects(source_table.the_geom, main_table.the_geom)
